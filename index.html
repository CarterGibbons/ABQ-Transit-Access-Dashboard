ABQ Transit Access Dashboard

This interactive tool visualizes census block-level social vulnerability and transit accessibility. Users can explore accessibility metrics through a choropleth map, select individual blocks, and view the corresponding 30-minute accessibility isochrone.

How to Use

  • Select a metric from the side panel.
  • Choose a time scenario.
  • Click a census block or search for an address.
  • Switch between choropleth and isochrone views as needed.

Data Sources

  • GTFS transit schedules
  • Road network data
  • U.S. Census block and tract data
  • CDC / Census-based social vulnerability measures
