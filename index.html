ABQ Transit Access Dashboard

About This Tool

This interactive dashboard visualizes transit accessibility and social vulnerability at the census block level in Albuquerque, New Mexico. Users can explore how access to key destinations varies across space and time and compare these patterns with neighborhood-level social vulnerability.

This tool visualizes the Albuquerque transit system before the 2025 ABQ RIDE Forward Network Plan changes. Accessibility is calculated using 2020 data. All data is for reference only and accuracy is not guaranteed.

How to Use - View tutorial video

Getting Started

Start by choosing an accessibility metric and a time period using the controls on the left side of the map.

The map will update to show how access changes depending on what you choose.

Reading the Map

The map uses color shading to show differences between places. darker areas mean higher values, while lighter areas mean lower values.

For accessibility maps, darker areas usually indicate that more opportunities can be reached within 30 minutes. For the Social Vulnerability Index (SVI), darker areas indicate higher vulnerability.

Exploring a Location

  • Click a census block to view its values and charts.
  • Switch between Data View and 30-Minute Travel Area.
  • Search for an address to select its corresponding block automatically.

Reading the Charts

    When a block and accessibility metric is selected:

  • The Top Chart shows the accessibility values for that block across time, with the selected time period highlighted.
  • The Bottom Chart shows how all census blocks are distributed based on their Social Vulnerability Index (SVI) and the currently selected accessibility metric. Each shaded hexagon represents a group of blocks with similar values. The selected block is shown as a dot so you can see how it compares to the overall pattern.

Accessibility Metric Explanation

Social Vulnerability Index (SVI)

The Social Vulnerability Index (SVI), developed by the Centers for Disease Control and Prevention (CDC), indicates the relative level of social vulnerability based on demographic and socioeconomic conditions.

SVI is measured on a 0–1 scale, where 0 indicates lower vulnerability and 1 indicates higher vulnerability.

Accessibility Metrics

  • Jobs Access: Employment opportunities reachable within 30 minutes by transit and walking.
  • Healthcare Access: Healthcare destinations certified to provide Medicaid-approved care reachable within 30 minutes by transit and walking.
  • School Access: APS public and charter schools reachable within 30 minutes by transit and walking.
  • Food Access: Food-related destinations that accept SNAP benefits, excluding convenience stores, reachable within 30 minutes by transit and walking.

Higher accessibility values indicate a greater number of reachable opportunities within the 30-minute threshold.

Calculation Methods

Accessibility is calculated using a 30-minute travel-time threshold that combines walking and scheduled transit travel. This analysis is conducted using ArcGIS Network Analysis Tools.

All accessibility metric data is geocoded and aggregated to census blocks.

For each census block and time period, the analysis identifies which destinations can be reached within that threshold and summarizes reachable opportunities by type.

The travel area view displays the corresponding 30-minute isochrone for the selected block and selected time period.
